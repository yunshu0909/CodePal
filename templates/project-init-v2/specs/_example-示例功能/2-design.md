# 2 · 设计 — 深色模式开关（示例）

> 本项目当前无全局设计规范，以下契约自洽——这正是「零设计系统下写 2-design」的样子。

## 设计需求总结

- 设置页加一个开关项「深色模式」，默认关。
- 切换即时生效（无需刷新），全站配色切换。
- 选择持久化，下次打开保持。
- 状态：开 / 关两态；无加载 / 错误态（纯本地）。

## 实现契约

- **数据**：`theme: 'light' | 'dark'`，存 localStorage，key = `app-theme`。
- **应用方式**：在根元素加 / 去 `data-theme="dark"`，CSS 变量按该属性切换（不逐组件改色）。
- **模块职责**：
  - `theme.js`：读 / 写 localStorage + 切根属性，导出 `getTheme()` / `setTheme(t)`。
  - 设置页：一个开关，`onChange` 调 `setTheme`。
  - 应用启动：读 localStorage 应用一次。
- **例外**：无。
